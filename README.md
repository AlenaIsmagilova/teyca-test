# Teyca

##### Реализована часть интерфейса для работы с клиентской базой и PUSH-рассылками: авторизация, просмотр клиентов и отправка PUSH - уведомлений.

<!-- ### [Демо](https://alenaismagilova.github.io/test-task-only/) -->

### Стек

1. Angular 19, TypeScript 5
2. Rxjs 7
3. scss

### Быстрый старт

1. Клонирование и установка зависимостей

```markdown
git clone https://github.com/AlenaIsmagilova/teyca-test.git
```

```markdown
cd teyca-test
```

```markdown
ng build
```

2. Запуск проекта

```markdown
ng serve
```

3. Откройте в браузере http://localhost:4200

4. Используйте любые логин и пароль для входа в приложение, сервер в любом случае вернет тестовый токен.
